## Projeto

  Você pode baixar o projeto em formato ZIP.
  
### Clonar Projeto
  
   ```bash
$ git clone https://github.com/ngustavo80/letmeask.git
```  

### Instalar dependências

  ```bash
$ yarn
```
  
### Iniciando aplicação

  ```bash
$ yarn start
```
